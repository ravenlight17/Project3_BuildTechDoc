Java Project 3: Introduction to Objects

Object-Oriented-Programming (OOP) is a fundamental style of programming based on the notion of “objects” and “classes”. OOP combines data and the procedures that operate that data, together in a unit. To access or modify that data, you need to use those procedures (methods) within that object. This style or philosophy helps to construct manageable applications composed of simple, reusable components. It certainly is not the only style of programming, but it is a popular design and architectural strategy employed by professional programmers and system architects. It is important that you become familiar with OOP since it is used in Android, Java, C++, Python, JavaScript and other programming environments.

You want to help people find the smartphone of their dreams in your smartphone store. You have a current inventory of existing Android phones, iPhones, and even some ancient not-so-smart phones. A user has an idea of what smartphone features they want. They want to search your store to see if you have them in stock. Let’s write a small app that displays the smartphone inventory in your store. Many real-world programming languages follow Object-Oriented-Programming (OOP) principles. It is important for you to have a thorough understanding of OOP beyond simple classes and objects that we covered in a previous lesson. Understanding OOP principles is essential to use objects and classes efficiently in a successful project.
